Integração Telegram

Receba alertas de falhas e relatórios automáticos a cada 3 horas diretamente no seu Telegram.

Configuração do Bot

Obtenha em @BotFather
Use @userinfobot para descobrir

Alertas Automáticos

Falha no envio Notificação imediata quando uma mensagem ou arquivo falhar
Sessão desconectada Alerta quando um número cair e precisar escanear o QR novamente
Erro crítico na API Alerta quando o serviço interno travar ou crashar
Relatório a cada 3 horas Status das sessões, mensagens enviadas, arquivos e erros

Como configurar

1
Abra o Telegram e busque @BotFather
2
Envie /newbot e siga as instruções
3
Copie o Token gerado e cole no campo ao lado
4
Adicione o bot ao grupo/canal desejado e envie uma mensagem
5
Use @userinfobot para descobrir o Chat ID
6
Cole o Chat ID, salve e clique em Testar Conexão
O bot só consegue enviar mensagens em grupos onde foi adicionado como membro.
