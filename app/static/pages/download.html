Como instalar

  1. Baixe o instalador ZapDinAgentSetup.exe
  2. Execute como Administrador (botão direito → "Executar como administrador")
  3. Cole o Token de Ativação da empresa (gerado em Sistema → Token API)
  4. Aguarde o download do Chromium (~3 min na primeira vez)
  5. O serviço ZapDinAgent sobe automaticamente — pronto

Como funciona

1. Cliente acessa o painel ZapDin via navegador (rodando na nuvem)
2. Painel envia comandos → Agent local recebe via WebSocket seguro
3. Agent controla WhatsApp Web no Chromium local (posto do cliente)
4. Cada empresa só vê seus próprios dados (multi-tenant por token)

Atualizações automáticas

O ZapDin Agent verifica novas versões a cada 6 horas e atualiza sozinho. Não é preciso reinstalar.

Requisitos

  • Windows 10/11 x64
  • ~1 GB de espaço em disco
  • Conexão com internet (download inicial do Chromium)
  • Permissão de Administrador (instalar serviço Windows)
