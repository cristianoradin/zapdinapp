Download — ZapDin Agent
Instalador Windows do ZapDinAgent — software que roda na máquina do posto e conecta WhatsApp Web local ao painel cloud.
ZapDin Agent v—
Windows 10/11 x64 · ~15 MB · Single-exe install
Baixar instalador personalizado (recomendado)
Token desta empresa já vem embutido. Cliente só executa o .bat (Administrador) — sem digitar nada.
Baixar .exe genérico (precisa colar token manual)

Como instalar

  1. Baixe o instalador (recomendado: botão verde)
  2. Execute como Administrador (botão direito → "Executar como administrador")
  3. Aguarde download do Chromium (~3 min primeira vez)
  4. Serviço ZapDinAgent sobe automaticamente — pronto

Atualizações automáticas

O ZapDinAgent verifica novas versões a cada 6 horas e atualiza sozinho. Não é preciso reinstalar.
